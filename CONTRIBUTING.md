# Contributing to MoodCompass

Thank you for your interest in contributing! This project aims to help people managing bipolar disorder, so thoughtful contributions are especially valued.

## How to Contribute

1. **Fork** the repository
2. **Create a branch** for your feature or fix: `git checkout -b feat/my-feature`
3. **Make your changes** and ensure they follow existing code style
4. **Test** that `npm run build` passes without errors
5. **Commit** with a clear message describing the change
6. **Open a Pull Request** against `main`

- Accessibility improvements
- Internationalization (i18n)
- Additional chart visualizations
- Test coverage

## Sensitivity Guidelines

This app serves people with mental health conditions. Please:

- Use compassionate, non-stigmatizing language
- Avoid clinical or diagnostic language in the UI
- Always include appropriate disclaimers with AI-generated content
- Never remove or hide crisis support resources
